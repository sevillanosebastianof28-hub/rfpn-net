# RFPN.NET

## About

RFPN.NET is a property development and investment platform connecting developers, investors, brokers, and professionals through structured workflows, verified parties, and AI-supported analysis.

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm i

# Start the development server
npm run dev
```
